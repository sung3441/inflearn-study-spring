# PORT

### 한 번에 둘 이상 연결해야 한다면?
#### port를 이용해서 같은 IP 내에서 프로세스를 구분할 수 있다.

ex) IP가 아파트라고 하면 port는 동, 호수라고 생각할 수 있다.
- 100.100.100.100:1122 => 게임
- 100.100.100.100:8080 => 메일 서버
- 100.100.100.100:10010 => 웹 브라우저 요청

#### PORT

- 0 ~ 65,535 할당 가능
- 0 ~ 1023: 잘 알려진 포트, 사용하지 않는 것이 좋음
  - FTP - 20, 21
  - TELNET - 23
  - HTTP - 80
  - HTTPS - 443
